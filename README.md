# Multipurpose AI Project (Next.js)

**Project Link:** [https://ai.rony.studio/](https://ai.rony.studio/)

This is a Next.js project where I connected various APIs like Gemini, DeepSeek, etc. Using this, you can create photos, write emails, generate content, and more. 

Since exceeding the API limits could lead to billing charges for me, I have just kept demo data in the live version for now.
